# Proyecto: Portafolio Web de Ricardo Garduño

Este proyecto contiene todo lo relacionado con Ricardo Garduño, incluyendo:

- Página web tipo currículum vitae (CV)  
- Secciones de inicio, acerca de mí, experiencia laboral, formación académica y contacto  
- Archivos de estilo (CSS) y scripts (JavaScript)  
- Recursos como imágenes, iconos y documentos  

El objetivo de este proyecto es presentar de manera clara y profesional la información personal y académica de Ricardo Garduño, además de servir como portafolio web personal.
